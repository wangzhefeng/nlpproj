# nlpproj

## Flow

* 数据语言(data language)
* 分词(tokenization)

## Task

* 命名实体识别(entity recognition)
* 词性标注(part-of-speech-tagging)
* 依存句法分析(dependency parsing)
* 句子分割(sentence segmentation)
* 文本分类(text classification)
* 词元化(lemmatization)
* 形态分析(morphological analysis)
* 实体链接(entity linking)
* ...
